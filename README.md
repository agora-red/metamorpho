# Metamorpho · Ágora Design Outputs

> The Element Man. Shape-shifts any brief into a production-grade HTML asset.

Public host for every output generated by the **Design Studio** Claude skill — decks, landings, wikis, Instagram sets, proposals, brand kits, emails. Every output is a folder with a self-contained `index.html`, published via GitHub Pages.

**Live index**: https://agora-red.github.io/metamorpho/

---

## How it works

1. Design Studio (Claude skill in `cyclone/.claude/skills/design-studio/`) generates an output.
2. On close, it commits the folder here and pushes.
3. GitHub Pages publishes it within ~30s at `https://agora-red.github.io/metamorpho/<slug>/`.
4. The homepage (`index.html`) reads `_manifest.json` and renders the gallery.

## Folder shape

```
<slug>/
├── index.html       # the deliverable
├── _briefing.md     # briefing that generated it
├── _edits.jsonl     # iteration log
└── assets/          # images, fonts, data
```

## Manifest

`_manifest.json` is the canonical index. New outputs are **prepended** to the `outputs` array:

```json
{
  "type": "deck | instagram | proposal | wiki | brand-kit | landing | one-pager",
  "typeLabel": "Deck | Instagram | Proposal | Wiki | Brand kit | Landing | One-pager",
  "title": "Short title",
  "description": "What it solves · 1-2 lines",
  "detail": "N slides · subcategoría  |  feed · carousel · story  |  1-page",
  "date": "MES YYYY",
  "href": "./<slug>/index.html",
  "slug": "<slug>"
}
```
